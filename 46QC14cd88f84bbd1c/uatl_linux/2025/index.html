Cette activité Linux Découverte est une activité pratique, par conséquent, vous allez manipuler. Et pour cela, vous avez plusieurs possibilités. Vous pouvez venir :

  1. avec un PC portable sous Linux

  2. avec un Mac sous MacOS

  3. avec un PC portable à recycler

  4. avec un PC portable sous Windows que vous souhaitez conserver intact

  5. avec un PC portable sous Windows que vous souhaitez conserver

  6. les mains dans les poches

Avec un PC portable sous Linux

Vous possédez déjà un PC portable sous Linux, c’est parfait, venez avec.

Avec un Mac sous MacOS

MacOS est un Unix, vous pouvez donc apprendre plein de choses (le fonctionnement du système, manier la ligne de commande) avec cette activité.

Avec un PC portable à recycler

Vous avez un PC portable à "recycler", typiquement un matériel ancien qui ne supporte plus Windows (architecture 64 bits quand-même). alors vous apprendrez à installer Linux sur le disque local. Le disque local sera reformaté, donc penser à sauvegarder les données avant de venir si besoin.

Avec un PC portable sous Windows que vous souhaitez conserver en l’état

Votre portable est sous Windows, et vous y tenez : pas question de toucher a son disque local. Vous chargerez alors un système Linux à partir d’une clé USB. Le disque dur local reste intact, tout se passe en mémoire vive. L’inconvénient, c’est que vous perdez tout votre travail à la fin de chaque séance (puisque tout est volatile), et les temps de réponse ne sont pas terribles (les temps d’accès d’une clé USB sont médiocres). Si vous optez cette option, alors venez vendredi avec une clé USB (4Go minimum) qui sera reformatée.

Avec un PC portable sous Windows que vous souhaitez conserver

Installez WSL, le sous-système Windows pour Linux sur votre Windows. Inconvénient : n’ayant pas de PC Windows sous la main, je ne pourrai pas vous guider. Mais cette option est intéressante, et si vous êtes plusieurs dans ce cas, on formera un petit groupe de travail.

Les mains dans les poches

Cette option est identique à l’avant-dernière, à la différence que les TP se font sur un poste de travail de l’UATL. Un système Linux est à charger en début de chaque session depuis une clé USB. Mêmes incovénients : vous perdez tout votre travail à la fin de la session, et les temps de réponse ne sont pas terribles. Si vous optez cette option, alors venez vendredi avec une clé USB (4Go minimum) qui sera reformatée.
