Linux découverte, aspects pratiques

Jacquelin Charbonnel assisté de Claude Biczysko et Bruno Menan.
Vous vous demandez si cette activité est pour vous, alors voici quelques explications.

L’activité Linux découverte est orientée pratique. Pendant les séances, vous allez donc manipuler sur ordinateur. Et pour cela, vous avez plusieurs possibilités. Vous pouvez venir :

  1. avec un PC portable sous Linux,

  2. avec un PC portable à recycler,

  3. avec un PC portable sous Windows,

  4. les mains dans les poches.

PC portable sous Linux

Vous possédez déjà un PC portable sous Linux, c’est parfait, venez avec.

PC portable à recycler

Vous avez un PC portable ancien, qui ne supporte plus Windows (architecture 64 bits quand-même), et vous voulez lui donner une seconde vie. Profitez de cette activité pour installer Linux sur son disque local. Le disque sera reformaté à la première séance, penser donc à sauvegarder les données avant si nécessaire.

PC portable sous Windows

Si votre version de Windows le permet, le mieux est d’installez WSL, le sous-système Windows pour Linux. Vous aurez un Linux dans votre Windows. Inconvénient : n’ayant pas de PC Windows sous la main, je ne pourrai pas vous guider. Mais cette option est intéressante, et si vous êtes plusieurs dans ce cas, on formera un petit groupe de travail.

Autre solution, si vous ne souhaitez pas toucher à votre Windows, vous pouvez démarrer votre PC à partir d’une clé USB contenant un système Linux. Dans ce cas, vous n’utilisez ni votre Windows, ni son disque, tout se passe en mémoire vive. L’inconvénient, c’est que vous perdez votre travail à la fin de chaque séance (puisque tout est volatile), et les temps de réponse ne sont pas terribles (les temps d’accès d’une clé USB sont médiocres). Si vous choisissez cette option, alors venez à la première séance avec une clé USB (4Go minimum) qui sera reformatée.

Les mains dans les poches (conditionné à la salle attribuée à l’activité)

Cette option est identique à la précédente, si ce n’est que les TP se font sur un poste de travail de l’UATL. Un système Linux est chargé en début de chaque session depuis une clé USB. Mêmes inconvénients : vous perdez votre travail à la fin de la session, et les temps de réponse ne sont pas terribles. Si vous choisissez pour cette option, alors venez à la première séance avec une clé USB (4Go minimum) qui sera reformatée.
